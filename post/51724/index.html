plan B

Upside Down
2017-10-10

从昨天开始，搭载着 SS 的 VPS IP 大规模被 GFW 封锁，昨天换了一个 IP 以后还是能够继续使用。

不过今天（2017.10.10），GFW 再次升级，不只是单纯的封禁 IP，而是能够在保证不封锁 IP 的情况下，将 SS 流量封锁，不论何种加密方式。所以 VPS 上的 SS 已经全部都挂掉了。

幸好的是 SSR 暂时还能用，不过谁知道好日子还能延续多久呢。

为防止 SSR 也遭遇封锁，所以服务器上已经搭建好了 plan B - GFW.Press。

所以大家就先去GFW.Press下载好客户端，以备不时之需。
